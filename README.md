PocketPal Ultra
A retro-inspired virtual pet simulator built for the web. Take care of your digital companion, manage your daily budget, and watch your pet evolve based on how you treat it.

Features
Pet Evolution: Choose your starting companion and watch it grow through three life stages (Baby, Teen, and Adult).

Budget Management: Unlike standard pet sims, you have to manage a wallet. Allocate funds for food, fun, and health—if you run out of money, your pet’s health will suffer.

Dynamic Needs: Keep track of Hunger, Energy, Social, and Fun meters in real-time.

Minigames: Earn rewards and boost your pet's mood by playing the built-in Mole-grid game.

Performance Reports: Get a final letter grade based on your caretaking skills and financial responsibility.

How to Play
Name your pet and select a species (Nebula Cat or Solar Pup).

Set your budget. Decide how much of your $200 starting cash goes toward specific categories.

Interact. Click the action buttons to feed, clean, or play with your pet.

Survive. Keep your pet's health above zero for as many days as possible.

Technical Details
This project is a Single File Application. Everything—the logic, the styling, and the structure—is contained within one HTML file for easy portability.

Language: Vanilla JavaScript (ES6+)

Styling: Custom CSS with CSS Variables for centralized design tokens.

Assets: Uses standard unicode characters and Google Fonts (Nunito and Fredoka One).

Installation and Setup
Since this is a client-side web app, there is no installation required.

Download index.html.

Open it in any modern web browser like Chrome, Firefox, or Safari.

To host it online, simply upload the file to GitHub Pages.
